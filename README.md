🔧 L.P. TechOS — Sistema de Gestão de Oficina (ERP)
O L.P. TechOS é um sistema de gestão de serviços de assistência técnica (ERP) desenvolvido para otimizar o fluxo de trabalho de oficinas, desde a entrada de equipamentos até a finalização do reparo, integrando métricas de produtividade e gamificação para a equipe.

🚀 Funcionalidades Principais
Fila de Serviços: Gestão dinâmica de Ordens de Serviço (Entrada, Andamento, Conclusão).

Painel Administrativo: Visão consolidada de faturamento e produtividade da equipe.

Gestão de Perfis: Configuração personalizada de técnicos, avatares e especialidades técnicas.

Gamificação Motivacional:

Meta Coletiva: Barra de progresso semanal para engajamento da equipe.

Ranking Top 3: Pódio automatizado com recompensas (Bônus, iFood, etc).

Dashboard Visual: Gráficos interativos de performance (Tempo de Operação vs. Pausas).

🛠️ Tecnologias Utilizadas
Front-end: HTML5, CSS3 (Responsivo), JavaScript (Vanilla - Sem dependências externas).

Back-end: PHP 8+ para lógica do servidor e manipulação de arquivos.

Armazenamento: JSON (Gerenciamento de dados leve e portátil).

Design: Interface minimalista, organizada e voltada para a eficiência do técnico.

💻 Como Rodar o Projeto
Certifique-se de ter um ambiente local com PHP instalado (ex: XAMPP, Laragon ou servidor local).

Clone este repositório para a pasta do seu servidor (ex: htdocs ou www).

Garanta que o arquivo db.json tenha permissão de escrita para que o sistema possa salvar as Ordens de Serviço e perfis.

Acesse index.php no seu navegador.

📈 Visão Técnica
O diferencial deste projeto é a aplicação de lógica de negócios e UX (User Experience) no desenvolvimento. Além do CRUD de serviços, o sistema foi desenhado para atuar como uma ferramenta de gestão real, utilizando:

Cálculos Dinâmicos: Processamento de dados em tempo real para gerar o Ranking da Equipe.

Visualização de Dados: Implementação nativa de gráficos (Canvas API) para análise de produtividade, sem a necessidade de bibliotecas pesadas de terceiros.

Arquitetura: Código modular e organizado para facilitar manutenções futuras.
